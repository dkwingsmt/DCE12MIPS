# DCE12MISP/asmblr

A simple (but yet functional) MIPS assembler in C++. 

May compile directly using

    $ cmake .

while the command below (out-of-source) is highly recommended:

    $ mkdir debug
    $ cd debug
    $ cmake ..
Or use the equivalant script:

    $ bash build -d

For more options, see `bash build -h`.
